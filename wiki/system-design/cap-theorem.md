# CAP Theorem

CAP Theorem (Also known as Brewer theorem) is a set of pricinciples that guide us in trade-offs between three key components of a distributes systems.

- __Consistence__: All nodes need to see the same data.
- __Availability__: The system needs to stay available even though hidden fail occurs or in case os request peak.
- __Partition Tolerance__: The system needs to continue working even though a specific partition fail.
