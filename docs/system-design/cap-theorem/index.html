CAP Theorem

CAP Theorem (Also known as Brewer theorem) is a set of pricinciples that guide us in trade-offs between three key components of a distributes systems.

  • Consistence: All nodes need to see the same data.
  • Availability: The system needs to stay available even though hidden fail occurs or in case os request peak.
  • Partition Tolerance: The system needs to continue working even though a specific partition fail.
